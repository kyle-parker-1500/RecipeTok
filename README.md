# RecipeTok
- This app runs off of a local API that can be found at [https://github.com/kyle-parker-1500/RecipeAPIForRecipeTok](https://github.com/kyle-parker-1500/RecipeAPIForRecipeTok)
- Instructions for running the API are found in its readme
- To get the app up and running correctly you have to restart the database by cleaning the app and rebuilding it
- Make sure that you have the API running before you do this
- Finally, run the app, it should take up to a couple minutes to load at the maximum
